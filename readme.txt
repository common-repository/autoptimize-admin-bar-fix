== Description ==

Fixes issue with Autoptimize plugin. This plugin will show your admin bar on frontend again.
Will fix the bar for normal users in a next version.

== Installation ==

1. Upload directory `autoptimize-admin-bar-fix` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!
